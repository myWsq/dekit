Toolbar Controls

Responsive toggle On/off — when off, iframe fills available space (current behavior)
Device dropdown Preset devices: iPhones, iPads, Android phones, common desktops
Width × Height inputs Editable — typing custom values switches to "Custom" device
Rotate button Swaps width ↔ height
Zoom selector "Auto fit" calculates scale to fit canvas area; manual: 50%–150%
DPR label Read-only, shows device pixel ratio from preset
Touch cursor Toggle — when on, iframe shows a circle cursor simulating touch
